# 🛒 Blinkit Sales Dashboard | Microsoft Excel

## 📌 Project Overview

The Blinkit Sales Dashboard is an interactive Microsoft Excel dashboard designed to analyze sales performance, customer ratings, product categories, outlet performance, and business KPIs. The dashboard enables users to monitor sales trends and gain actionable business insights through interactive visualizations.

---

## 🎯 Objective

To build an Excel-based Business Intelligence dashboard that converts raw sales data into meaningful insights for business performance analysis and decision-making.

---

## 🚀 Key Features

- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Dynamic Slicers
- Sales Performance Analysis
- Outlet Performance Analysis
- Product Category Analysis
- Customer Rating Analysis

---

## 📊 Dashboard Insights

The dashboard includes:

- Total Sales
- Average Sales
- Number of Items Sold
- Average Customer Rating
- Outlet Size Analysis
- Outlet Location Analysis
- Fat Content Analysis
- Item Type Analysis
- Year-wise Outlet Establishment
- Sales Distribution
